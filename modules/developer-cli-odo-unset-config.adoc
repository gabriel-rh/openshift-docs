= Unsetting a value

You can unset a value for a preference key by using the following command:

.Command syntax
[source,terminal]
----
$ odo preference unset <key>
----

[NOTE]
====
You can use the `-f` flag to skip the confirmation.
====

.Example command
[source,terminal]
----
$ odo preference unset updatenotification
----

.Example output
[source,terminal]
----
