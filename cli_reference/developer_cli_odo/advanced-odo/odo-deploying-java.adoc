= Deploying with Java Spring Boot
include::_attributes/common-attributes.adoc[]
:context: odo-deploying-java

To deploy your Java Spring Boot application:

. Containerize your application by creating a Dockerfile.
. Modify your `devfile.yaml` to add your Kubernetes configuration.
. Run `odo deploy`.
